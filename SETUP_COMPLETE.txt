üöÄ TO START THE APPLICATION
================================================================================

1. CONFIGURE GHL API KEY (REQUIRED):
   Edit .env file and add your GoHighLevel API key:
   GHL_API_KEY=your_actual_api_key_here

2. START REDIS (REQUIRED):
   Option A - WSL: wsl -> sudo apt-get install redis-server -> redis-server
   Option B - Windows: Download from https://github.com/microsoftarchive/redis/releases
   Option C - Docker: docker run -d -p 6379:6379 redis

3. START FLASK APP:
   python run.py

4. START WORKERS (in new terminal):
   python worker.py

üìä WHAT THE SYSTEM DOES
================================================================================

1. Companies register with their GHL location ID
2. Companies upload leads (single, CSV, or bulk API)
3. Leads are queued for background processing
4. Workers send leads to GoHighLevel with proper tagging
5. GHL triggers SMS campaigns based on tags
6. Dashboard shows upload statistics and success rates
